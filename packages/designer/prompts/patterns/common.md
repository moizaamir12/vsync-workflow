# Common Workflow Patterns

## 1. Fetch → Transform → Display
API call → data transformation → UI display. Most common pattern.

## 2. Barcode Scanning (Mobile)
Camera scan → API lookup → form update → save

## 3. Form Data Collection
UI form → validation → API submission

## 4. Data Pipeline
Fetch → filter → sort → aggregate → display/send

## 5. AI Processing
Input → AI agent → display results

## 6. Scheduled Task
Cron trigger → fetch → process → notify
